# Íná

##
Wo íná!

##
Ínán jo.

##
Íná dáno.

##
Íná n fun ígbóná.

##
Íná n fun ni ímólé.

##
Wo, íná!

##
Íná jẹ iyanú.

##
O si làgbàrà.
