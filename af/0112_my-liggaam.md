# My liggaam

##
Ek kan daarmee hardloop.

##
Ek kan daarmee spring.

##
Ek kan daarmee dans.

##
Ek kan daarmee swem.

##
Ek kan daarmee touspring.

##
Ek kan daarmee skop.

##
Ek kan daarmee vryspring.

##
Maar ek kan nooit daarmee vlieg nie.
